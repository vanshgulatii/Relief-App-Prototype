
# 🌿 Relief — Grow Calm

An immersive, scroll-driven web experience that connects **mental wellness** with a **living forest**.

Relief is an experimental storytelling interface where users “grow” a real tree in the Black Forest through interaction, blending **nature, IoT concepts, and mindfulness** into a cinematic digital journey.

---

## ✨ Features

* 🌌 Dynamic sky transitions (night → dawn → daylight)
* 🌳 Procedural tree growth based on scroll
* 🌠 Particle systems (stars, fireflies, rain)
* 🌙☀️ Moon to sun transition
* 🎵 Sound wave visualization (tree music concept)
* 💧 Water interaction simulation
* 📊 Animated impact stats
* 🧠 Scroll-based storytelling panels
* 📱 Fully responsive canvas rendering

---

## 🧠 Concept

“Every scroll is a breath. Every interaction, a moment of calm.”

Relief explores:

* Digital interaction linked to real-world nature
* Mindfulness through micro-interactions
* Storytelling using animation and motion design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (animations, gradients, layout)

---

## 🎮 How It Works

1. User scrolls the page
2. Scroll progress (0 → 1) controls:

   * Sky color transitions
   * Tree growth
   * Particle effects
   * Scene changes
3. Panels fade in/out based on scroll position
4. Final section presents a call-to-action
